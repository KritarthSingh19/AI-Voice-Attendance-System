# AI-Powered Multimodal Biometric Attendance System

## Project Vision

Build a secure attendance platform that verifies a user's identity using both voice biometrics and facial biometrics before marking attendance.

The goal is to combine Machine Learning, Computer Vision, Audio Processing, Analytics, and Software Engineering into a complete real-world AI application.

This project serves as my flagship AI/ML portfolio project.
